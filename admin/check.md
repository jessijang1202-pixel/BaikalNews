# 바이칼뉴스 AI 기사 체크리스트

AI 어시스턴트 집필실(주제 입력 / 링크 재구성 / 오늘의 화제 기사 / 정보성 기사 추천)로 초안을 생성할 때마다,
이 체크리스트를 기준으로 AI가 자기 점검 결과를 함께 보여줍니다. 이 점검은 참고용 보조 장치이며,
**최종 발행 여부는 항상 최상락 또는 장승희의 검토·승인을 거쳐야 합니다.** (편집 규약 제7조)

새 점검 항목이 필요하면 아래 형식(`- [ ] 항목 설명`)에 맞춰 자유롭게 추가/수정하십시오.
다음 기사 생성부터 자동으로 반영됩니다.

## 1. 인간 저널리즘 원칙 준수
- [ ] 과도하게 균형 잡힌 상투적 결론("~라고 볼 수 있다", "귀추가 주목된다" 등 모호한 클리셰)의 반복이 없다
- [ ] 문장 길이와 구조에 자연스러운 리듬과 변주가 있으며, 기계적으로 정형화된 패턴이 반복되지 않는다
- [ ] 구체적인 취재원, 인용, 현장감 있는 디테일이 최소 1개 이상 포함되어 막연한 일반론에 그치지 않는다
- [ ] 단순 정보 재배열이 아니라 기자의 관점과 통찰이 드러난다

## 2. 논조 스타일 일치도 (민들레 / 오마이뉴스 등 지정 스타일)
- [ ] 선택한 스타일에 학습된 문체 규칙이 실제 문장에 반영되었다
- [ ] 헤드라인의 어조가 해당 매체 특유의 표현 방식과 닮아 있다 (예: 오마이뉴스 - 인용형·대화체, 민들레 - 날카로운 은유와 단정적 문장)
- [ ] 글의 관점과 맺음말이 해당 논조(권력 비판, 약자 연대, 개혁 지향 등)에 부합한다

## 3. 사실관계 및 취재 윤리
- [ ] 확인되지 않은 수치·날짜·인명·기관명을 단정적으로 서술하지 않았다
- [ ] 원천 기사(링크 재구성·오늘의 화제 기사)를 그대로 베끼지 않고 다른 각도로 독자적으로 재구성했다
- [ ] 특정 개인이나 단체에 대한 근거 없는 명예훼손성 표현이 없다

## 4. 저널리즘 구조 완성도
- [ ] 헤드라인 - 리드 - 본문(소제목 2개 이상) 구조를 갖췄다
- [ ] 카테고리가 기사 내용과 실제로 일치한다
- [ ] SEO 타이틀 · 메타 설명 · 슬러그가 기사 내용을 정확히 반영한다

## 5. 정보성 기사 전용 점검 (정책지원금 · 세금 · 복지 주제인 경우)
- [ ] 신청 대상 · 방법 · 마감일 등 확정되지 않은 정보를 단정적으로 서술하지 않고, 관계 기관 확인을 안내했다
- [ ] 독자가 실제로 행동에 옮길 수 있을 만큼 구체적인 정보를 담았다

## 6. 검색엔진 최적화 (SEO: 네이버 · 다음 · 구글)
- [ ] 제목 앞부분에 핵심 키워드가 자연스럽게 포함되어 있다
- [ ] 검색 노출용 제목(SEO 타이틀)과 실제 기사 제목의 의미가 사실상 동일하며 낚시성으로 다르지 않다 (네이버·다음은 제목 불일치를 감점 요인으로 봄)
- [ ] 리드 문단 첫 2~3문장 안에 핵심 키워드가 자연스럽게 포함되어 있다
- [ ] 본문 소제목(H2)에 핵심 키워드 또는 연관 키워드가 포함되어 있다
- [ ] 메타 설명이 120~155자 내외로 핵심 키워드를 포함하면서도 자연스럽고 클릭을 유도한다
- [ ] 키워드를 부자연스럽게 반복(키워드 스터핑)하지 않았다
- [ ] URL 슬러그가 핵심 키워드를 반영한 짧고 명확한 영문/로마자 조합이다

## 7. 편집 및 발행 절차 준수
- [ ] 이 초안은 아직 "임시저장(draft)" 상태이며 발행되지 않았다
- [ ] 편집기로 전송 후 반드시 최상락 또는 장승희의 검토·승인을 거쳐야 발행됨을 인지하고 있다
