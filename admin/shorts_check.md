# 바이칼뉴스 숏폼(쇼츠) 대본 체크리스트

대본 자동 생성(Step 2) 직후, 이 체크리스트를 기준으로 AI가 자체 점검 결과를 함께 보여줍니다.
이 점검은 참고용 보조 장치이며, 통과하지 못한 항목이 있으면 후킹 문구나 컷 대본을 직접 수정하거나
"2. 대본(스크립트) 자동 생성"을 다시 눌러 재생성하십시오.

새 점검 항목이 필요하면 아래 형식(`- [ ] 항목 설명`)에 맞춰 자유롭게 추가/수정하십시오.
다음 대본 생성부터 자동으로 반영됩니다.

## 1. 후킹 (0:00~0:03) — 가장 중요, 스크롤을 멈추게 하는가
- [ ] 첫 3초 안에 시청자의 스크롤을 멈추게 할 만큼 강력하다 (의외의 사실, 반전, 숫자, 강한 질문, 갈등 중 하나 이상)
- [ ] "~에 대해 알아봅니다", "~ 소식입니다" 같은 밋밋하고 뻔한 일반론이 아니다
- [ ] 15자 내외로 짧고, 한 호흡에 읽히는 임팩트 있는 문장이다
- [ ] 기사 전체를 요약하는 문장이 아니라, 기사에서 가장 자극적이거나 의외인 지점 하나만 정확히 찌른다
- [ ] 상단 배너 제목(topBarTitle)도 후킹 문구와 겹치지 않으면서 그 자체로 시선을 끈다

## 2. 전반부 (0:00~0:08, Veo 영상 또는 업로드 자료 대체)
- [ ] Veo 프롬프트가 후킹 문구와 같은 장면/감정을 시각적으로 뒷받침한다 (후킹과 동떨어진 장면이 아니다)
- [ ] 8초 안에 소화 가능한 단일하고 명확한 장면 하나로 묘사되어 있다 (여러 장면을 욱여넣지 않는다)

## 3. 후반부 각 컷의 대본(나레이션)과 자막 (0:08~0:30)
- [ ] 각 컷의 대본(나레이션)이 자막을 단순 반복하지 않고, 자막보다 더 설명적이고 자연스러운 문장이다
- [ ] 각 컷의 자막이 15자 내외로 짧고, 나레이션의 요약이 아니라 그 자체로 임팩트 있는 독립적인 문구다
- [ ] 컷들을 순서대로 읽었을 때 같은 내용을 반복하지 않고 기승전결이 느껴지는 자연스러운 흐름이다
- [ ] 마지막 컷이 여운을 남기거나 다음 행동(전체 기사 확인, 공유, 댓글 등)을 자연스럽게 유도한다

## 4. 전체 구조 및 사실관계
- [ ] 30초 전체가 하나의 완결된 이야기로 읽힌다 (중간에 맥락이 끊기거나 갑자기 다른 화제로 넘어가지 않는다)
- [ ] 원본 기사의 핵심 사실관계(수치, 인명, 날짜 등)를 왜곡하거나 과장하지 않았다
- [ ] scriptMd(전체 대본 문서)에 후킹 문구가 실제로 포함되어 있고 강조되어 있다 (생성되었지만 문서에는 반영되지 않는 경우가 있다)

## 5. 인물·언어 규칙 (이미지/영상 생성 공통 규칙 재확인)
- [ ] 등장인물 묘사가 있다면 한국인/동양인 외모로만 묘사되어 있다 (외국인/서양인/혼혈 묘사 없음)
- [ ] 화면에 텍스트가 등장하는 프롬프트가 있다면 한글로만 묘사되어 있다 (영어 등 외국어 지시 없음)
